                                                                             Bash (Bourne Again Shell)
1)Bash was developed in 1989 by Brian Fox.
2)Bash was created as a free replacement for the Bourne shell. It quickly became the default for GNU and many Linux systems.
3)Over time, Bash added features from other shells, such as the Korn shell (KSH) and C shell (CSH), and it became a very versatile tool for command-line users.


4)Bash is used to run commands and write scripts on Unix/Linux systems.
5)Bash is the default shell on most Linux and macOS systems.


6)Bash is a powerful tool for developers and system administrators.
7)Understanding and mastering Bash is important for working in Unix/Linux.

/*	****here is the depth concept based on the  ""'CHMOD'"" command****

>> In Unix-like systems, every file has three types of permissions for three categories of users:

"Category"	"Meaning"
 u	         user (owner)
 g	         group
 o	         others (everyone else)
 a	         all (user + group + others)

And each has three types of permissions:

"Symbol"	"Meaning"
 r	         read
 w	         write
 x	         execute (run)




>>  chmod u+x script.sh         # Give execute permission to the user (owner)
    chmod go-w file.txt         # Remove write permission from group and others
    chmod a=r file.txt          # Set read-only for all (user, group, others)


>> | Permission  | Value |
   | ----------- | ----- |
   | read (r)    | 4     |
   | write (w)   | 2     |
   | execute (x) | 1     |

>> 7 = read + write + execute

   6 = read + write

   5 = read + execute

   4 = read only


>> example :  >> chmod 755 script.sh

>> 7 → user: read + write + execute

   5 → group: read + execute

   5 → others: read + execute
	*/

while loop : 
           Syntax:
                  Initialization
                  while [ condition ];
                  do
                  commands
                  ((increment/decrement))
                  done



Example : 
         1)count=1
           while [ $count -le 3 ];
           do
           echo "Count is $count"
           ((count++))
           done
           
           
             
         2)n=5
           while [ $n -le 10 ];
           do
           echo "Count is : $n"
           ((n++))
           done



Until loop :  Until loops are similar to while loops, but they execute until a specified condition becomes true. 
            Syntax : 
                     Initialization
                     until [ condition ];
                     do
                     commands
                     ((increment/decrement))
                     done



            Example(1) : 
                     n=1
                     until [ $n -ge 3 ];
                     do
                     echo "the count is : $n"
                     ((n++))
                     done

                     output :-   the count is : 1
                                 the count is : 2



            Example(2) : 
                     n=1
                     until [ $n -gt 3 ];
                     do
                     echo "the count is : $n"
                     ((n++))
                     done

                     output :-   the count is : 1
                                 the count is : 2
                                 the count is : 3



Break and Continue : Break and continue statements are used to control loop execution. break exits the loop, while continue skips to the next iteration.

These statements are typically used inside conditional blocks to alter the flow of the loop.

Example on  "Continue" : 
                        for i in {1..5};
                        do
                        if [ $i -eq 3 ];
                        then
                        continue
                        fi
                        echo "Number $i"
                        done

Example on "Break" : 
                     for i in {1..5};
                     do
                     if [ $i -eq 4 ];
                     then
                     break
                     fi
                     echo "Number $i"
                     done

Example on both "Continue" and "break" : 
                                         for i in {1..5};
                                         do
                                         if [ $i -eq 3 ];
                                         then
                                         continue
                                         fi
                                         echo "Number $i"
                                         if [ $i -eq 4 ]; then
                                         break
                                         fi
                                         done   



Nested Loops : Nested loops allow you to place one loop inside another, enabling more complex iteration patterns.

Each loop must be closed with its own done.



Syntax : 
         for var1 in list1;
         do
         for var2 in list2;
         do
         commands
         done
         done



Example : 
         for i in {1..3};
         do
         for j in {1..2};
         do
         echo "Outer loop $i, Inner loop $j"
         done
         done

                         Syntax :
